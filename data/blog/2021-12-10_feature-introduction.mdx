# 博客的功能介绍

## 功能

采用 mdx 而非 md 格式，可以在 mdx 中写 jsx 语法来实现更加自由的样式编写。

### 标题名

```
---
title:
date:
lastmod:
tags:
thumbnail:
hero:
summary:
---
```

现在需要的参数有
- title: 文章标题
- date: 创建时间
- lastmod: 最后修改时间
- tag: 标签
- thumbnail: 文章目录中显示的图片
- hero: 文章顶部图片
- summary: 总结

:::info
thumbnail 图片推荐大小为 650 x 300

hero 图片推荐大小为 1920 x 1000
:::

### 引用

:::info
这是一个通知类型的引用
:::

通过 ::: 包裹住产生一个引用块，在 ::: 后面填写 none、info、warning、error 来展示四种样式

```mdx
:::info
这是一个通知类型的引用
:::
```

### 代码块

```java {3,13-15} showLineNumbers
public class SingletonTest {

    public static void main(String[] args) {
        System.out.println(Singleton.INSTANCE.getTest());
        System.out.println(Singleton.INSTANCE.getTest());
    }

    public enum Singleton {
        INSTANCE;

        private final Test test;

        Singleton() {
            test = new Test();
        }

        public Test getTest() {
            return test;
        }
    }

    private static class Test {

    }
}
```

通过语言，{} 包裹要高亮的行（可不写），以及 `showLineNumbers` 显示行号（可不写）

````
```java {3,13-15} showLineNumbers
```
````

- [x] 已完成事项
- [ ] 代办事项1
- [ ] 代办事项2
